# Вітаємо!

Саме тут ми коротенько розповімо, як завантажити та зайти до гри в перший раз. Уважно читайте та повторюйте всі кроки — і у вас гарантовано все вийде!

---

## 🛒 Що потрібно для початку

Для початку необхідно мати **придбану версію GTA 5** у **Steam** або **Epic Games**.

Після чого потрібно **встановити [RAGEMP](https://rage.mp)** з офіційного сайту.

Також не забудьте **зареєструватись або увійти в свій SocialClub**.

---

## 🔍 Як знайти сервер

Після встановлення, потрібно зайти в RAGEMP та знайти сервер. Це можна зробити двома способами:

1. **Через пошук:**  
   Введіть у пошукову строку:  
   ```
   LocalRP  
   ```

2. **Напряму за адресою сервера:**  
   ```
   s1.localrp.com.ua:22005
   ```

---

## 🧾 Реєстрація акаунту

Після підключення — **зареєструйте акаунт**, і все — ви можете грати!

На спавні вас буде чекати **квестовий NPC Стефан**, який допоможе вам розібратись та освоїтися в новому місці.

> 📝 **Наполегливо рекомендуємо** пройти початковий квест — саме там ви дізнаєтесь базові навички.

📖 Детальніше про квест — у **наступній статті**.
